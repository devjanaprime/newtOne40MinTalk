Intro & overview
================
* non-violent
* create happiness
* levels shouldn't be too large (about 3-5 min for experienced players)
* highly replayable (parrots, notes, achievements, narrative unlocks, etc)
* multiple platforms (console launch first XBOne, hopefully)
